- Learn how to authenticate and make your first API call on  **Waves** API. Waves  provide speech-AI feature like  _TTS_ (Text to speech) and  _STT_ (Speech to text).

### Endpoint & Method

```bash
POST https://waves-api.smallest.ai/v4/health
```

### Required Headers

- _Authorization_ : Bearer  YOUR_API_KEY
- _Content-Type_:  application/json
- _Accept_:  application/json

### Sample Request

<CodeGroup>

```python python
import requests

url = "https://waves-api.smallest.ai/v4/health"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json",
    "Accept": "application/json"
}

response = requests.post(url,headers=headers)
print(response.json())
```


```bash curl
curl -X POST "https://waves-api.smallest.ai/v4/health" \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-H "Accept: application/json"
```

</CodeGroup>

<Note>
  Replace YOUR_API_KEY with your  API key from WAVES dashboard.
</Note>

### Sample Response

```json
{
  "status": "ok",
  "message": "API connection successful"
}
```

- A succesfull response confirms that your API key is valid and the wave service is reachable
